# 🎲 다이스 가디언즈 (Dice Guardians)

랜덤 유닛을 소환하고, 합성하여 강화하고, 밀려오는 25 웨이브의 적을 막아내는 **랜덤 디펜스** 게임입니다.

## 🎮 플레이 방법

1. **소환** (🎲 버튼) — 50G를 사용하여 랜덤 유닛을 소환합니다
2. **합성** — 같은 등급의 유닛 두 개를 드래그하여 겹치면 상위 등급으로 합성됩니다
3. **배치** — 유닛을 드래그하여 슬롯 간 이동할 수 있습니다
4. **판매** — 유닛을 판매 영역으로 드래그하면 25G를 회수합니다
5. **스킵** — 웨이브 대기 시간을 건너뛰고 20G 보너스를 받습니다

## 🗡️ 유닛 종류 (5종)

| 유닛 | 특징 |
|------|------|
| 🗡️ 전사 | 근접 공격, 크리티컬 + 스턴 |
| 🏹 궁수 | 원거리, 빠른 공속, 연사 + 관통 |
| 🔮 마법사 | 범위 공격, 스플래시 + 둔화 |
| 💚 서포터 | 인접 유닛 버프 (ATK/공속/범위) |
| ⚡ 특수 | 적 디버프 (둔화/방어감소/독/빙결) |

## ⭐ 등급 시스템

- ⭐ 커먼 (55%) → ⭐⭐ 레어 (30%) → ⭐⭐⭐ 에픽 (12%) → ⭐⭐⭐⭐ 레전드 (3%)
- ⭐⭐⭐⭐⭐ 미시크는 **합성으로만** 획득 가능

## 🌊 웨이브

- 총 25 웨이브 (매 5웨이브마다 보스 등장)
- 적 유형: 일반 🟢 | 빠름 🔵 | 탱크 🔴 | 비행 🟡 | 보스 💀

## 🎵 특징

- **Web Audio API**로 생성된 사운드 이펙트 (외부 에셋 없음)
- 모바일 최적화 (터치 드래그, 세로 모드)
- 일시정지 (ESC / ⏸ 버튼)
- 속도 조절 (1x / 2x)
- 로컬 최고 기록 저장

## 🛠️ 기술 스택

- **Phaser 3** — 게임 엔진
- **TypeScript** — 타입 안정성
- **Vite** — 빌드 도구
- **Web Audio API** — 사운드 (외부 파일 없음)

## 🚀 개발

```bash
npm install
npm run dev      # 개발 서버 (localhost:3000)
npm run build    # 프로덕션 빌드 → dist/
```
